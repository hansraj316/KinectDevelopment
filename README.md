# KinectDevelopment (Legacy)

Legacy experiments built around the Microsoft Kinect SDK era.

## Buildability notes

- Expected toolchain: Visual Studio 2010/2012 era on Windows.
- Requires legacy Kinect SDK/runtime versions that are no longer actively maintained.
- Modern systems may need compatibility mode or virtualization for reproducible builds.

## Compatibility matrix

| Component | Expected Version |
| --- | --- |
| OS | Windows 7/8 vintage |
| IDE | Visual Studio 2010/2012 |
| SDK | Kinect for Windows SDK (legacy) |
